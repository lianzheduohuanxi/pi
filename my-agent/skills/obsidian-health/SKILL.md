# 健康记录技能

当用户提到健康、身体、睡眠、休息、药物、症状等词汇时使用此技能。

## 步骤

1. 如果用户提供了足够的健康相关信息，直接使用 `obsidian_record` 工具以 "health" 分类记录
2. 如果信息不够明确，简短询问用户需要记录什么健康信息
3. 如果用户提供了详细数据，一并记录
4. 确认记录成功

## 示例

- "昨晚睡了 8 小时" → 直接记录: "睡眠: 8 小时"
- "头疼" → 直接记录: "症状: 头疼"
- "吃了感冒药" → 直接记录: "药物: 感冒药"

## 记录格式

```
健康事项: 具体描述
```
