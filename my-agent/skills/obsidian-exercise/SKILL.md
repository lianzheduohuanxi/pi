# 运动记录技能

当用户提到运动、锻炼、跑步、健身、体力活动时使用此技能。

## 步骤

1. 如果用户提供了足够的信息（运动类型），直接使用 `obsidian_record` 工具以 "exercise" 分类记录
2. 如果信息不够明确，简短询问运动类型和时长
3. 如果用户提供了数据（距离、次数、重量），一并记录
4. 确认记录成功

## 示例

- "跑了5公里" → 直接记录: "跑步 5km"
- "做了30分钟瑜伽" → 直接记录: "瑜伽 30分钟"
- "举铁，深蹲80kg 5组" → 直接记录: "力量训练: 深蹲 80kg x 5组"

## 记录格式

```
运动类型: 详细描述 [时长] [强度] [备注]
```
