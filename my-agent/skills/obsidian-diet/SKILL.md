# 饮食记录技能

当用户提到食物、用餐、饮食、吃了什么时使用此技能。

## 步骤

1. 如果用户提供了足够的信息（吃了什么），直接使用 `obsidian_record` 工具以 "diet" 分类记录
2. 如果信息不够明确（比如只说"吃了"但没说什么），简短询问具体内容
3. 如果用户提供了详细信息（热量、营养），一并记录
4. 确认记录成功

## 示例

- "我吃了早餐" → 询问吃了什么，然后记录
- "今天午饭吃了火锅" → 直接记录: "午餐: 火锅"
- "下午吃了个苹果" → 直接记录: "加餐: 苹果"

## 记录格式

```
餐次: 食物描述 [热量估算] [营养备注]
```
