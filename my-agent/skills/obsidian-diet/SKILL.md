# 饮食记录技能

当用户提到食物、用餐、饮食、吃了什么时使用此技能。

## 步骤

1. 如果用户提供了足够的信息（吃了什么），使用 `quicklog` 记录（自动分类到 diet）
2. 如果信息不够明确（比如只说"吃了"但没说什么），简短询问具体内容
3. 如果用户提供了详细信息（热量、营养），一并记录

## 示例

- "今天午饭吃了火锅" → `quicklog("🍽️ 午饭吃了火锅")`
- "下午吃了个苹果" → `quicklog("吃了个苹果")`
- "早餐吃了面包和牛奶" → `quicklog("🍽️ 早餐: 面包和牛奶")`

## 注意

- `quicklog` 会自动分类、更新 streaks、检查成就
- 不需要手动调用 `obsidian_record`
