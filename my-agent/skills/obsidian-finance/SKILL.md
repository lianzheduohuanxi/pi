# 财务记录技能

当用户提到钱、花钱、收入、支出、购买、消费、工资、账单等词汇时使用此技能。

## 步骤

1. 如果用户提供了足够的财务信息，直接使用 `obsidian_record` 工具以 "finance" 分类记录
2. 如果信息不够明确，简短询问用户需要记录什么财务信息
3. 尽量获取金额、用途、时间等详细信息并记录
4. 确认记录成功

## 示例

- "花了 50 元买咖啡" → 直接记录: "支出: 50 元 (咖啡)"
- "收到工资了" → 直接记录: "收入: 工资"
- "交了电费 200 元" → 直接记录: "支出: 200 元 (电费)"

## 记录格式

```
收入/支出: 金额 (用途)
```
