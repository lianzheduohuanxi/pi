# 财务记录技能

当用户提到钱、花钱、收入、支出、购买、消费、工资、账单等词汇时使用此技能。

## 步骤

1. 如果用户提供了足够的财务信息，使用 `quicklog` 记录（自动分类到 finance）
2. 如果信息不够明确，简短询问用户需要记录什么财务信息
3. 尽量获取金额、用途等详细信息并记录

## 示例

- "花了 50 元买咖啡" → `quicklog("💰 支出 50 元 (咖啡)")`
- "收到工资了" → `quicklog("💰 收入: 工资")`
- "交了电费 200 元" → `quicklog("💰 支出 200 元 (电费)")`

## 注意

- `quicklog` 会自动分类、更新 streaks、检查成就
- 不需要手动调用 `obsidian_record`
