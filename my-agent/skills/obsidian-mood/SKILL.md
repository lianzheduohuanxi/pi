# 心情记录技能

当用户提到心情、情绪、感受、开心、难过、焦虑等词汇时使用此技能。

## 步骤

1. 如果用户提供了足够的信息（当前心情状态），使用 `quicklog` 记录（自动分类到 mood）
2. 如果信息不够明确，简短询问用户当前心情
3. 如果用户提供了详细信息（心情持续时间、原因、表现等），一并记录

## 示例

- "今天心情很好" → `quicklog("😊 心情很好")`
- "有点焦虑" → `quicklog("😰 有点焦虑")`
- "感觉很平静，因为完成了重要的工作" → `quicklog("😊 平静，完成了重要工作")`

## 注意

- `quicklog` 会自动分类、更新 streaks、检查成就
- 不需要手动调用 `obsidian_record`
